# vue2-storybook

Demo repo for lightening talk on using Vue 2 and Storybook 6.

## Project setup

```
yarn install
```

### Compiles and hot-reloads web app for development

```
yarn run serve
```

### Compiles and minifies web app for production

```
yarn run build
```

### Run storybook

```
yarn run storybook
```

### Lints and fixes files

```
yarn run lint
```

### Run end-to-end tests

```
yarn run test:e2e
```

### Run unit tests

```
yarn run test:unit
```
